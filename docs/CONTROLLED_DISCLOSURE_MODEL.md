# Controlled Disclosure Model

The project uses a staged disclosure model.

## Public

Public material is descriptive and non-executable. It explains purpose,
boundaries, and accountability without publishing reconstructive detail.

Public material may also support citizen science education and stewardship
literacy when it stays non-operational and does not imply approval to build,
deploy, or operate private systems.

## Private Review

Private review may include implementation details, private documents, or
technical evidence when the reviewer has an appropriate role, need, and review
context.

## Release Approval

Publication of any reduced technical excerpt requires review for:

- rights and ownership;
- privacy;
- security;
- dual-use risk;
- user safety;
- operational maturity;
- misleading or overbroad claims.

## Default Rule

When a detail could materially help reconstruct the private build, it remains
private unless explicitly approved for release.
