It exists so the line can say plainly which invariants must never break even
as later growth laws, action bases, and compositions evolve above them.

## A0 Floor

`A0` is the non-breakable floor.

These axioms are the minimal set that must remain true for the active slice to
stay coherent.

### A1 Layer Integrity

A thing must be treated as the kind of knowing it is.

- roots are not definitions
- definitions are not context
- context is not procedure

No cross-layer masquerading.

### A2 Root Identity Invariance

Once a root is anchored, its identity does not change under allowed
transformations.

Trunks do not mutate.

### A3 Downward Grounding

Every higher layer must resolve downward to roots.

No floating structures.

### A4 Discernment Admissibility

Only that which passes discernment may enter or remain.

Admission is earned, not assumed.

### A5 Non-Inflation

No object may carry hidden content from a higher layer.

Each layer stays clean.

### A6 Translation Preservation

All translations must preserve:

- layer identity
- anchor identity
- dependency structure

Mapping must not collapse structure.

### A7 Posture Stability

The acting system must:

- hold without premature action
- recover from drift
- refuse forced procedural collapse

Action is gated by stability.

## Constitutional Effect

If `A0` fails, the slice loses coherence.

If `A0` holds, later laws may grow on top of it without rewriting the floor.

All future laws must preserve `A0`.

This note does not define the full law set. It defines the minimum floor that
all later doctrine, contracts, and actions must obey.

## Working Summary

`A0` is the minimal invariant floor for this slice: layer integrity, root
identity invariance, downward grounding, discernment admissibility,
non-inflation, translation preservation, and posture stability.
