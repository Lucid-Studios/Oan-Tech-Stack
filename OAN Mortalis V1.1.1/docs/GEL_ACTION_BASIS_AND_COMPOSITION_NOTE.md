## Minimal Action Basis

The first action basis is:

`A0 = { Attend, Reduce, Decompose, Discriminate, Anchor, Relate, Retain, Refuse, Recover }`

The nine actions are:

- `Attend`
  - isolate a candidate without altering it
- `Reduce`
  - strip definitional, contextual, and procedural inflation
- `Decompose`
  - split a mixed candidate into more discernible parts
- `Discriminate`
  - determine what kind of knowing the candidate is
- `Anchor`
  - bind a root candidate to an existing or new anchor
- `Relate`
  - form lawful relation without identity collapse
- `Retain`
  - admit a lawful object into `GEL`
- `Refuse`
  - deny admission or continuation when law fails
- `Recover`
  - return the acting system to prime posture after drift

Action basis and composition operators are contract-backed while canonical
flows remain doctrine-only.

## First Composition Algebra

The first composition algebra is:

`C0 = { Sequential, Guarded, Choice, Iteration, RecoveryWrapper, RefusalTerminal }`

The composition operators are:

- `Sequential`
  - perform the second action only after the first action succeeds
- `Guarded`
  - perform the next action only when the guard holds
- `Choice`
  - select exactly one lawful branch
- `Iteration`
  - repeat until fixed point or terminal routing
- `RecoveryWrapper`
  - wrap a flow so posture returns to prime
- `RefusalTerminal`
  - terminate flow with no further composition
