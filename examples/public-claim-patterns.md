# Example: Public Claim Patterns

These examples are non-executable wording patterns.

## Safer Claim

The project studies governance patterns for accountable AI systems, including
human review, safety boundaries, disclosure control, and evidence-limited
claims.

## Overbroad Claim To Avoid

The project publishes a complete system for reproducing the private runtime.

## Safer Boundary Statement

Detailed architecture, datasets, source code, build steps, and runtime payloads
are withheld pending private review.

## Overbroad Boundary Statement To Avoid

All meaningful technical details are public and ready for independent
implementation.
