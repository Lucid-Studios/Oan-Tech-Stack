# Support

## Use The Right Channel

For release-shell or public-boundary clarity:

- open a public boundary correction;
- open a public explanation request.

For security-sensitive or reconstruction-sensitive disclosures, do not use the
public issue tracker. Follow [SECURITY.md](SECURITY.md).

## Repository Scope

This repository is currently a bare release shell. It does not provide build
support, runtime support, deployment support, model configuration support,
archived-material restoration on demand, or private implementation assistance.
