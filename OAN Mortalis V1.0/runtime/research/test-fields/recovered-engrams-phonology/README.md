# Recovered Engrams and Phonology Test Field

## Purpose

This lane preserves a recovered internal experiment family for bounded inspection and later mining.

It is intentionally:

- lineage-bearing
- audit-friendly
- non-authoritative

## Layout

- `preserved/archives/`
  - raw recovered zip bundles copied into the repo
- `preserved/texts/`
  - recovered text artifacts copied into the repo
  - raw preservation takes priority over transcription cleanup in this pass
- `preserved/oan-moral-test-set/`
  - copied summary/verification CSVs only
- `extracted/`
  - expanded contents of the small recovered zip bundles for direct inspection
- `manifest.json`
  - tracked inventory with hashes, sizes, and metadata-only references

## Boundary

This field is not current doctrine and does not grant authority to historical metrics or schemas.

Use it for:

- lineage tracing
- bounded experiment recovery
- test-field seeding

Do not use it for:

- silent contract replacement
- governance bypass
- admission authority
