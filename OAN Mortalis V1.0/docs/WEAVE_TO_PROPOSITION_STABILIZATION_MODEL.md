# WEAVE_TO_PROPOSITION_STABILIZATION_MODEL

## Purpose

This document defines the doctrine by which the active `OAN Mortalis V1.0` stack moves from pre-instantiated coherence into proposition-bearing stabilization.

It exists to prevent one architectural collapse:

- treating formative coherence as though it were already an ordinary module surface or governed identity

This document is doctrine only. It does not claim runtime implementation of the stages it names.

## Governing Warning

> The system must not confuse formative coherence with governed identity.
>
> What emerges may be structurally real before it is propositionally stable, propositionally stable before it is morphism-capable, and morphism-capable before it is admissible to Sanctuary custody.

## Invariants

- generated structure != accepted truth
- formed candidate != governed identity
- latent weave != accountable surface

## Ordered Chain

The ordered chain for this doctrine is:

1. pre-instantiated coherence
2. proposition-bearing stabilization
3. admissible proposition surface
4. Sanctuary-governed accountability

Morphism-bearing stabilization or candidacy is a later threshold. It does not occur before proposition-bearing stabilization.

## 1. Pre-Instantiated Coherence

Pre-instantiated coherence names the latent formative weave from which localizable surfaces may later emerge.

At this stage:

- the weave is structurally real but not yet proposition-bearing
- the weave is not yet an object among objects
- coherence precedes stable self-readable surfaces
- Sanctuary is not yet the source of this coherence

Within this regime:

- `OE` and `cOE` are continuity-binding conditions
- `OE` and `cOE` must not be flattened into ordinary stack modules
- `OE` and `cOE` must not be described as already-instantiated root substrate

This stage is not yet:

- proposition stabilization
- morphism capability
- governed identity
- custody mutation

## 2. Proposition-Bearing Stabilization

Proposition-bearing stabilization is the first threshold at which a stable, inspectable, repeatable proposition-bearing formation can appear.

At this threshold:

- a formation may become proposition-bearing
- a formation may become inspectable and repeatable
- a formation may become suitable for proposition-level compile

But it is still true that:

- proposition-bearing structure is not yet accepted truth
- proposition-bearing stabilization is not yet morphism capability
- proposition-bearing stabilization is not yet governed identity

Within this regime:

- `SelfGEL` and `cSelfGEL` are the first stabilized self-readable surfaces that emerge inside the binding or coherence regime
- `SelfGEL` and `cSelfGEL` must not be treated as root substrate
- proposition compile is the first explicit cognitive compile layer

## 3. Admissible Proposition Surface

Not every stabilized proposition is admissible.

Admissibility is the threshold where proposition-bearing structure becomes fit for:

- observation
- review
- governance intake
- possible downstream handling

An admissible proposition surface is:

- proposition-bearing
- inspectable
- reviewable
- fit for governance intake

An admissible proposition surface is not yet:

- governed identity by default
- custody mutation by default
- proof of lawful cross-frame transport

## 4. Sanctuary-Governed Accountability

Sanctuary does not generate formative coherence.

Sanctuary does not define the weave.

Sanctuary accepts only what has become:

- proposition-bearing
- observable
- admissible
- accountable

This means Sanctuary is the accountability boundary, not the source of pre-instantiated coherence.

Governed identity remains distinct from:

- mere formation
- proposition-bearing stabilization
- admissible proposition surface

And governed identity is not yet custody mutation by default.

## Relationship To Current Stack Reading

This doctrine preserves the following active interpretation:

- CradleTek hosts conditions
- `OE/cOE` are continuity-binding conditions, not ordinary modules
- `SelfGEL/cSelfGEL` are the first stabilized self-readable surfaces inside that coherence regime
- Lisp/SLI is the nearest computational analogue to the formative weave itself
- proposition stabilization occurs before morphism candidacy
- Sanctuary governs only what has localized enough to become proposition-bearing, observable, admissible, and accountable
