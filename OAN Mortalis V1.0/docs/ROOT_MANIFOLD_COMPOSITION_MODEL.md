# ROOT_MANIFOLD_COMPOSITION_MODEL

## Purpose

This document defines the first bounded composition law for root-native symbolic structures.

It exists to keep the atlas from becoming a symbolic dictionary with loose compositional freedom. The canonical `RootAtlas` is instead treated as a manifold of interlacing predicate cores with lawful refinement, bridge, and constructor relations.

## Root Core Ownership

Each predicate root owns:

- one canonical root key
- one canonical atlas entry
- one canonical root-native symbolic anchor
- one lawful refinement position inside the atlas

Prefix and suffix operators may contextualize a root core.
They may not replace the root core as the invariant anchor.

## Composition Layers

The first bounded manifold composition model distinguishes:

1. root core
   - invariant predicate anchor
2. refinement relation
   - lawful parent/child or adjacent root relation
3. constructor body
   - prefix/root/suffix structural composition
4. domain bridge
   - explicit cross-domain semantic relation
5. rendered symbolic form
   - optional merged artifact only

This means a merged glyph is never the source of composition truth.

## Lawful Composition

Lawful root manifold composition requires:

- root core remains explicit
- constructor operators are admitted by symbolic domain policy
- refinement edges are explicit or absent
- bridge domains are declared, not inferred
- reserved-domain collisions are rejected

## Exploratory Composition

The first build may admit exploratory but non-canonical compositions only when:

- they remain in experimental extension space
- they do not collide with governance/meta or disciplinary reserved domains
- they are not mistaken for canonical atlas-owned structures

Exploratory composition is therefore bounded, not free-form.

## Prohibited Composition

The following are prohibited in canonical atlas composition:

- heuristic best-match domain bridging
- merged render forms standing in for structural composition
- disciplinary symbol reuse without explicit bridge policy
- experimental extension shapes silently entering canonical atlas space

## Relation to Engram Formation

The atlas manifold exists before Engram closure.

That means:

- roots provide lawful landing coordinates
- constructor bodies provide lawful symbolic contextualization
- refinement and bridge law provide admissible composition boundaries
- only then may `EngramDraft` bind and seek closure

This keeps atlas composition prior to governed memory movement.
