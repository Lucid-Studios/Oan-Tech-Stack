# IUTT_TRANSFORMATION_BOUNDARY_LAW

## Purpose

This note defines the `IUTT` boundary for `Sanctuary.GEL` intake.

It exists to fix the lawful interval in which symbolic transformation may
occur before membrane landing and without pretending that transformation is
already runtime consequence.

## Governing Compression

> `IUTT` is the lawful interval space in which admissible symbolic
> transformation may occur.
>
> transformation is not execution, closure, or authorization.

## `IUTT` Office

`IUTT` is the bounded interval between predicate-candidate formation and
membrane landing.

Its office is to govern:

- lawful symbolic re-expression
- bounded transformation passage
- continuity-preserving reform
- explicit retention of transformation witness

It is not:

- `RootAtlas`
- runtime law
- service authorization
- direct procedure execution

## Required Transformation Fields

Every `IUTT` transformation event should carry at minimum:

- `iutt_interval_id`
- `source_candidate_ref`
- `transformation_class`
- `continuity_relation`
- `origin_universe`
- `target_universe`
- `interval_witness`

## Allowed Transformation Classes

The minimum classes in this phase are:

- `symbolic-preserving`
- `relation-preserving`
- `scope-bounded`
- `refusal-bearing`

No transformation class in this phase may imply runtime obligation.

## Boundary Rules

`IUTT` transformation may occur only when:

- source candidate structure is explicit
- symbol classification is complete enough for the requested interval
- continuity relation is named
- the requested transform does not exceed candidate-bearing standing

## Refusal Law

The following must force refusal, hold, or non-promotion:

- interval transform without witness
- transform that hides symbol loss
- transform that crosses universes without lawful mapping
- transform that implies procedure or runtime by semantic elegance alone

## Explicit Non-Grants

This note does not grant:

- proposition promotion by interval passage alone
- procedure standing
- runtime law
- service authorization
- active cryptic metabolism

## Working Summary

`IUTT` now stands as the lawful interval between predicate formation and
membrane landing.

That interval permits bounded symbolic transformation without letting
transformation impersonate execution or authority.
