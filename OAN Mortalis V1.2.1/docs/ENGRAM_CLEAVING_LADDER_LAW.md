# ENGRAM_CLEAVING_LADDER_LAW

## Purpose

This note defines the lawful semantic ascent for `Sanctuary.GEL` intake in
`V1.2.1`.

It exists to keep pre-Lisp and pre-code intake from collapsing into later
symbolic metabolism, later code carriage, or runtime authority.

## Governing Compression

> intake may preserve, classify, map, transform, and encode for cryptic use;
> it may not promote, mint, or activate.
>
> not every proposition may become procedure.
> not every procedure may become active runtime law.

Semantic formation layers are not constructor classes.

The next interior-awareness cluster derives above this intake ladder without
changing the non-runtime boundary.

## Scope

This note governs `Sanctuary.GEL` intake before `SLI.Lisp`, later C#
carriage, service binding, or runtime activation.

It does not grant:

- predicate promotion
- engram minting
- service authorization
- active governing `CME` standing
- runtime activation

## Cleaving Ladder

The lawful ascent is:

`root -> predicate -> proposition -> procedure -> master`

Each layer carries a distinct office.
No layer may counterfeit the office of a later layer.

### `root`

`root` is the first admissible semantic unit after verbatim and symbolic
preservation.

It may include:

- lexical anchors
- symbolic anchors
- canonical terms
- stems and variants
- operator markers
- root-family members

It is not:

- a proposition
- a procedure
- active runtime meaning

### `predicate`

`predicate` is the first relation-bearing structure formed from one or more
roots.

It may include:

- identity relation
- action relation
- modifier relation
- role relation
- domain-local semantic relation

It is not:

- closure-bearing truth
- a procedure
- authorization

### `proposition`

`proposition` is a witnessed semantic claim formed from predicate structure.

It may include:

- a bounded statement
- a declared relation
- a meaning event under a named universe binding

It is not:

- a procedure
- execution authority
- runtime law

### `procedure`

`procedure` is a governed action-bearing semantic structure formed from one or
more propositions.

It may include:

- action type
- preconditions
- refusal conditions
- transformation logic

It is not:

- active execution
- service authorization
- runtime law by default

### `master`

`master` is an integrated continuity-bearing knowledge construct formed from
admitted propositions and eligible procedures.

It may include:

- ancestry
- governing universe
- supporting propositions
- linked procedures
- witness chain
- continuity relation to other masters

It is not:

- a runtime activation grant
- a substitute for governance law
- proof of active life

## Transition Law

No transition may occur without:

- witness
- universe continuity or lawful universe mapping
- resolved symbol classification
- ancestry trace
- explicit refusal visibility when ascent is denied

The minimum transition order is:

- verbatim preservation
- symbol classification
- root mapping
- predicate formation
- `IUTT`-bounded transformation
- membrane landing
- engram-bearing encoding

## Retention Classes

The intake ladder may emit only these retention classes in this phase:

- `retained-as-witness`
- `retained-as-provisional-meaning`
- `retained-as-engram-ready`

`retained-as-witness` preserves form without granting meaning.
`retained-as-provisional-meaning` preserves structure without closure.
`retained-as-engram-ready` preserves admitted structure for later cryptic use
without minting or activation.

## Refusal Law

The following must force refusal, withholding, or non-promotion:

- root without symbol classification
- predicate with unresolved root ambiguity
- proposition without witness
- procedure attempting implicit execution
- master without ancestry trace
- any ascent that outruns the current lawful layer

## Explicit Non-Grants

This note does not grant:

- silent correction
- silent promotion
- engram minting
- runtime execution
- live `SLI.Lisp` authority
- code trust by semantic fluency alone

## Working Summary

`Sanctuary.GEL` intake now has a lawful cleaving ladder.

That ladder preserves a bright distinction between:

- semantic raw material
- structured relation
- declared meaning
- governed method
- integrated master knowledge

It also preserves the harder line beneath later work:

- proposition is not procedure
- procedure is not runtime law
