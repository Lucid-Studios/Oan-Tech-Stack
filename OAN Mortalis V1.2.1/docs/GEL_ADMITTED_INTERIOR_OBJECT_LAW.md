# GEL_ADMITTED_INTERIOR_OBJECT_LAW

## Purpose

This note defines the canonical at-rest semantic object for `Sanctuary.GEL`.

It exists to make lawful discernment terminate into one explicit interior
object rather than evaporating into transient process state or being silently
treated as promotion-ready.

## Governing Compression

> lawful discernment must end in a lawful resting object before any later
> ascent may be discussed.
>
> persistence inside `GEL` does not yet admit `Sanctuary.MoS` or
> `Sanctuary.cMoS` storage-seat standing without receipted handoff.

## Object Office

The only canonical resting object admitted in this phase is:

- `Held Interior Engram Object`

This object binds:

- propositions
- procedures
- active universe bindings
- correspondence edges
- contradiction edges
- posture binding
- continuity links
- witness chain

It is the at-rest termination object for lawful discernment in `GEL`.

## Formation Boundary

The `Held Interior Engram Object` may form only after:

- lawful intake
- lawful interior discernment
- explicit posture-bearing state
- continuity-bearing witness

It is not:

- a runtime object
- an agent
- a promotion candidate
- a storage-seat admission

## Canonical Fields

Every `Held Interior Engram Object` should carry at minimum:

- `held_id`
- `source_ingress_refs[]`
- `constructor_class`
- `active_universe_bindings[]`
- `proposition_refs[]`
- `procedure_refs[]`
- `correspondence_edges[]`
- `contradiction_edges[]`
- `ambiguity_state`
- `posture_binding`
- `continuity_links[]`
- `rest_state`
- `witness_chain[]`

## Explicit Non-Grants

This note does not grant:

- runtime object standing
- agentic standing
- promotion
- engram minting
- service authorization
- `Sanctuary.MoS` storage-seat standing
- `Sanctuary.cMoS` storage-seat standing
- active governing `CME` standing

## Working Summary

`Sanctuary.GEL` now has one canonical interior resting object.

That object is the lawful at-rest termination of discernment while remaining
inside `GEL`, non-promotional, and non-runtime.
