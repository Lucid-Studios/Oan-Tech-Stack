as active standing before CradleTek is constitutionally running.

active office.

This audit fails if any of the following occur:

- any implied active governing `CME` standing
- any implied CradleTek authorization from Sanctuary-side law
- any collapse of Mother/Father into Steward
- any loss of `CmePlacementWithheld`
- any silent move from prepared set/manifold into active office
