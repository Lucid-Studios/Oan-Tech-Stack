# GEL_SITUATIONAL_AWARENESS_FRAME_LAW

## Purpose

This note defines the first derived situational-awareness frame for
`Sanctuary.GEL`.

It exists to give `GEL` a lawful awareness surface without turning awareness
into autonomous action.

## Governing Compression

> situational awareness is a derived frame, not autonomous action.
>
> propositions tell `GEL` what the world seems to be; procedures tell `GEL`
> how it must behave toward that seeming.

Lawful awareness must terminate in a lawful resting object.

Awareness itself does not persist as action.

## Frame Office

The situational-awareness frame is formed from:

- propositions
- procedures
- correspondence and contradiction state
- posture law

It is a bounded local read.
It is not autonomous life, execution, promotion, minting, or runtime.

## Semantic And Governance Split

The frame must keep semantic condition distinct from governance condition.

Semantic condition includes:

- active universe candidates
- propositions
- correspondence and ambiguity
- absence signals
- malformation signals

Governance condition includes:

- procedures
- attack signals
- continuity state
- recommended posture

## Canonical Fields

Every situational-awareness frame should carry at minimum:

- `frame_id`
- `ingress_refs[]`
- `active_universe_candidates[]`
- `proposition_refs[]`
- `procedure_refs[]`
- `correspondence_refs[]`
- `ambiguity_set[]`
- `absence_signals[]`
- `malformation_signals[]`
- `attack_signals[]`
- `continuity_state`
- `recommended_posture`

## Derivation Rules

The frame may derive only from already-admitted lower surfaces in this phase.

It may not:

- infer execution authority
- overrule refusal law
- collapse contradiction into certainty
- treat posture as action

`recommended_posture` must remain one of:

- `admit`
- `provisional`
- `hold`
- `refuse`

## Explicit Non-Grants

This note does not grant:

- autonomous action
- runtime execution
- service authorization
- predicate promotion
- engram minting
- active governing `CME` standing

## Working Summary

`Sanctuary.GEL` now has a first lawful situational-awareness frame.

That frame is derived from semantic holdings, method holdings,
correspondence/contradiction state, and posture law while remaining below
runtime.
